# CheckOut_Proces
# PyTest Selenium Checkout Framework

## Project Overview
This project automates the checkout process using Selenium, Python, PyTest and Page Object Model.

## Features
- Login Automation
- Add to Cart
- Checkout Automation
- PyTest Framework
- POM Design Pattern
- Reusable Driver Setup

## Run Project
pytest -v -s
